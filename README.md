## Headline Projects

### [AccommodateME](https://github.com/AhmedAliQadir/AccomPro)
Multi-tenant SaaS platform for UK supported-housing and social-care providers. Manages residents, staff, incidents, compliance, properties, and documents in isolated per-organisation workspaces.

- **Stack:** React + TypeScript, TanStack Query, shadcn/ui, Vite | Node.js/Express, Prisma → Drizzle, PostgreSQL (Neon), JWT auth
- **Security:** AES-256-GCM document encryption, TLS 1.3, 4-tier RBAC, full audit logging, UK GDPR compliant
- **Status:** Built and pilot-ready

### [HMRC Tax Analyser](https://github.com/AhmedAliQadir/hmrc-tax-analyser)
UK tax analysis tool built with TypeScript. Calculates income tax, National Insurance, and take-home pay under current HMRC rules.
